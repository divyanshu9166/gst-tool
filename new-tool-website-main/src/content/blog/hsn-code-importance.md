Under the Indian Goods and Services Tax (GST) system, accurate classification of goods and services is crucial. This is achieved using the **Harmonized System of Nomenclature (HSN)** for goods, and the **Services Accounting Code (SAC)** for services. If you generate invoices, understanding how to use an [HSN Code Lookup](/hsn-code-lookup) tool is a mandatory compliance step.

## What is an HSN Code?

Developed by the World Customs Organization (WCO), the HSN is an internationally accepted method of classifying traded products. In India, the GST Council uses HSN codes to determine the exact tax rate applicable to a specific item. A typical HSN code can be 2, 4, 6, or 8 digits long, depending on the level of classification required.

For services, the GST regime utilizes SAC codes. Every service—from IT consulting to restaurant catering—has a designated 6-digit SAC code.

## Is it Mandatory to Mention HSN on Invoices?

Yes. As per the latest notifications from the [official GST portal](https://www.gst.gov.in/), mentioning HSN/SAC codes on tax invoices and in GSTR-1 returns is mandatory based on a business's Aggregate Annual Turnover (AATO):
- **Turnover up to ₹5 Crore**: 4-digit HSN code is mandatory for B2B invoices. (Optional for B2C invoices).
- **Turnover above ₹5 Crore**: 6-digit HSN code is mandatory for all invoices (B2B and B2C).

For import and export transactions, an 8-digit HSN code is mandatory regardless of turnover.

## Why Correct HSN Classification Matters

Using the wrong HSN or SAC code can lead to serious consequences:
1. **Incorrect Tax Rates**: Charging 5% instead of 18% because of a wrong code can result in massive tax liabilities and penalties during audits. (Use our [GST Calculator](/gst-calculator) to compute the exact amounts).
2. **Denial of Input Tax Credit (ITC)**: If the HSN code on your outward supply does not match the nature of your business, it can trigger scrutiny, potentially causing your buyer to lose their ITC.
3. **E-Way Bill Rejection**: An invalid HSN code will prevent you from generating an E-Way bill for the movement of goods.
