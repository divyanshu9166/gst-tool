## What is HSN Code in GST?

HSN stands for **Harmonized System of Nomenclature**. It is an internationally recognized system used to classify goods. When you export or import goods, the customs department uses this code to understand exactly what is in the container.

Under Indian GST, the government uses HSN codes to assign tax rates to specific products. If you sell goods, you use an HSN code. If you sell services, you use a **SAC (Services Accounting Code)**.

Every single invoice you generate must feature this code. If you file your GSTR-1 returns without it, the portal will throw an error, and your buyer will not get their Input Tax Credit (ITC).

## Understanding 6 Digit vs 8 Digit HSN Codes

The HSN system is structured hierarchically. The more digits a code has, the more specific the product description is.

- **2 Digits (Chapter):** Broad category (e.g., Chapter 61 is Articles of apparel).
- **4 Digits (Heading):** More specific (e.g., 6109 is T-shirts, singlets and other vests).
- **6 Digits (Sub-heading):** Highly specific (e.g., 610910 is Of cotton).
- **8 Digits (Tariff Item):** The most granular level used mainly for import/export.

**What do you need for domestic GST?**
As of 2026, businesses with an annual aggregate turnover of up to ₹5 crores in the preceding financial year must mention a minimum of **4-digit HSN codes** on B2B invoices. Businesses with a turnover exceeding ₹5 crores must use **6-digit HSN codes** on all invoices.

## How to Find HSN Code: Step-by-Step

Finding the correct code used to require flipping through a massive PDF provided by the CBIC. Today, an HSN code search should take less than five seconds.

1. **Identify the Core Product or Service:** Strip away brand names. If you sell an "Apple iPhone 16," your search term is "Mobile Phone." 
2. **Use a Search Tool:** Don't rely on outdated Google results from 2023, as many items were reclassified during the GST 2.0 update. 
3. **Verify the Tax Rate:** The HSN code determines the rate. Ensure the rate matches the new slabs (typically 5%, 18%, or 40%).

## How to Find SAC Code for Services

SAC (Services Accounting Code) works exactly like HSN but is exclusively for services. All SAC codes in India start with the number **99**.

For example, if you are a freelance web developer, you provide "Information technology (IT) design and development services." 
- The SAC chapter for services is 99.
- The major group for IT services is 83.
- The specific code for IT design is **998314**.

Under the new GST 2.0 structure, almost all services fall under the standard **18% GST slab**.

## Frequently Asked Questions

**Is it mandatory to show HSN code on an invoice?**
Yes. For B2B invoices, HSN codes are mandatory regardless of turnover. For B2C invoices, they are mandatory if your turnover exceeds ₹5 crore.

**What is the difference between HSN and SAC?**
HSN (Harmonized System of Nomenclature) is used to classify physical goods. SAC (Services Accounting Code) is used to classify services.

**Can I use a 4-digit HSN code instead of 6 digits?**
If your previous year's aggregate turnover was under ₹5 crore, you are only required to use a 4-digit HSN code for B2B supplies.

**What happens if I use the wrong HSN code?**
Using the wrong code constitutes a misdeclaration. It can lead to the application of an incorrect tax rate, resulting in a penalty of ₹50,000 or the tax amount evaded, whichever is higher.

**How do I find the HSN code for a completely new product?**
You must find the closest matching category based on the material the product is made from or its primary function. If in doubt, consult a tax professional or apply for an Advance Ruling.
